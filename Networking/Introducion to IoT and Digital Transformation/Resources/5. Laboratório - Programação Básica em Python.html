Parte 1: Pratique o básico do Python

Nesta parte, você aprenderá e praticará algumas Programação em Python.

Passo 1: Baixe e instale o Python3.

a. Navegue até www.python.org/downloads/ e baixe a versão mais recente para o seu sistema operacional.

b. Localize o arquivo baixado e instale o Python. Selecione Adicionar Python 3.x para PATH, onde x é a versão de lançamento. Adicionando Python aos tells PATH Windows onde você instalou o Python. Em seguida, clique em Instalar agora.

Você também pode instalar o Python 3.x usando a Microsoft Store. Pesquise Python e selecione a versão mais recente do Python.

Para a maioria das distribuições Linux, o Python já está instalado. Para instalar o IDLE no Ubuntu, digite o seguinte comando:

user@Ubuntu:~$ sudo apt-get install idle3

Para iOS, consulte https://www.python.org/download/mac/tcltk/.

Parte 2: Criar um script usando IDLE para Python

Nesta parte, você iniciará o IDLE e criará um script simples. IDLE significa Desenvolvimento Integrado e Ambiente de Aprendizagem. É suportado e incluído no pacote Python. Alguns dos principais recursos do IDLE para Python inclui estes:

=   Um shell Python (interpretador interativo) com coloração de entrada, saída e mensagens de erro.

=   Uma janela múltipla editor de texto com desfazer múltiplo, coloração Python, recuo inteligente, dicas de chamada, preenchimento automático e outros recursos.

=   A capacidade de pesquise em qualquer janela, substitua nas janelas do editor e pesquise por vários arquivos.

=   Um depurador com pontos de interrupção persistentes, revisão e exibição de namespaces globais e locais.

=   Configuração navegadores e outras caixas de diálogo.

Etapa 1: inicie o IDLE.

a. Inicie o IDLE no menu Iniciar do Windows.

Clique em Arquivo > Novo Arquivo para abrir um novo Python (sem título) roteiro.

c. No novo arquivo, digite print("Hello World!"), observe que os códigos são coloridos e os parênteses abertos e fechados são correspondentes.

Clique em Arquivo > Salvar, salve o script atual como 1.py no diretório Documentos (C:/Usuários/seu nome/Documentos). Clique no botão Salvar.

e. Clique em Executar > Executar Módulo (ou pressione F5). A janela do shell exibirá o resultado.

Passo 2: Crie seu próprio script.

Você pode não saber como criar um script Python complexo em o seu próprio. Mas você pode criar um script Python simples usando os exemplos no parte anterior.

Por exemplo, crie e salve o novo arquivo conforme 2.py em seu Diretório de documentos. Digite print(1+1) no novo arquivo e salve-o novamente. Execute o módulo e imprima os resultados de 1+1 e exiba no shell.

Experimente diferentes funções e você estará em sua maneira de usar uma linguagem de programação poderosa.
