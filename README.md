Cisco Networking Studies
This repository contains my practical work throughout the Cisco Networking Academy (NetAcad) program — a globally recognized curriculum that covers the fundamentals of networking, routing, switching, and network security, forming the foundation for the CCNA 200-301 certification.
All projects and labs are built using Cisco Packet Tracer, a powerful network simulation tool that allows real-world topology design and device configuration without physical hardware. The goal is not just to pass an exam, but to develop genuine hands-on understanding of how networks are designed, operate, and communicate.
---

The `Networking/` folder contains labs and projects organized by topic, covering both completed work and exercises currently in progress as I advance through the course.
