About the Course
The Cisco Networking Academy (NetAcad) is one of the most recognized networking programs in the world. This repository follows the CCNA track, which covers the foundations of networking, routing, switching, and security — skills required for the Cisco CCNA certification exam (200-301).
---
Goals
Complete the Cisco NetAcad curriculum
Pass the CCNA 200-301 certification exam
Apply networking fundamentals toward a career in network engineering and cybersecurity
